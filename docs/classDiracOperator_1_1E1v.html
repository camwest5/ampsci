Detailed Description

Electric dipole operator, v-form: \( \frac{ie}{\omega \alpha} \vec{\alpha}\).

\[ <a||d_v||b> = R \]

\[ R = -\frac{2e}{\omega \alpha} \int( f_ag_b <ka||s||-kb> - g_af_b <-ka||s||kb>)\,dr \]
